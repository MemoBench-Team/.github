# MemoBench

MemoBench is a benchmark for evaluating visual memory in world generation models.

- 🌐 Project page: https://memobench-team.github.io/
- 📄 Paper: https://arxiv.org/abs/2606.27537
- 🤗 Dataset: https://huggingface.co/datasets/tonyc54/MemoBench
- 💻 Code: https://github.com/MemoBench-Team/MemoBench
- 📊 Leaderboard: https://huggingface.co/spaces/tonyc54/MemoBench_Leaderboard
